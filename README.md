## Main Project

### LiteRaceSegNet: Road Damage Segmentation

LiteRaceSegNet is a lightweight semantic segmentation project for road-damage analysis.  
The project focuses on boundary representation degradation in road-damage segmentation and explores a lightweight Boundary-Guided Fusion architecture for improving boundary preservation.

Key areas:

- Road-damage semantic segmentation
- Lightweight CNN architecture
- Boundary-aware feature fusion
- CPU/GPU inference evidence
- Ablation study and threshold analysis
- Visual evidence review
